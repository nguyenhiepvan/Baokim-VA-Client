# Installation
```shell
composer require nguyenhiep/baokim-va-client
```

#Configuration

To configure the package you need to publish settings first:

```shell
php artisan vendor:publish --provider="Nguyenhiep\BaoKimVaClient\BaoKimVaClientServiceProvider"
```
